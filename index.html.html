RAG PDF Question & Answering System

Membangun sistem tanya jawab berbasis dokumen PDF menggunakan arsitektur Retrieval-Augmented Generation (RAG). Pengguna dapat mengupload PDF apapun, lalu mengajukan pertanyaan dalam Bahasa Indonesia — sistem akan menjawab berdasarkan konten dokumen secara real-time via REST API.

Alur sistem
📄 PDF Upload → Text Extraction → Chunking (500c) → Embedding (MiniLM) → FAISS Index
❓ User Query → Embed Query → Similarity Search → TinyLlama LLM → FastAPI Response

ARIMA vs LSTM for Automobile Production Forecasting with Multi-Interventions ✦ Jurnal Terindeks

Penelitian yang membandingkan model klasik ARIMA dengan deep learning LSTM dalam melakukan forecasting produksi mobil PT Astra Daihatsu Motor. Data mencakup 174 observasi bulanan (2010–2024) dengan tiga peristiwa intervensi besar: perubahan kebijakan 2013, pandemi COVID-19 (2020), dan peningkatan produksi SUV (2022).

Model Comparison · ARIMA vs LSTM

Perbandingan prediksi ARIMA dan LSTM terhadap data aktual produksi. LSTM lebih adaptif menangkap fluktuasi tajam, terutama pada periode pasca-2020.

Perbandingan MAPE dan RMSE model ARIMA vs LSTM pada data testing (Mei 2023 – Jun 2024). LSTM unggul di kedua metrik: MAPE 10.37% vs 11.59%, RMSE 4,587 vs 5,059.
